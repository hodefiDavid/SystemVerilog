Lab 1 on importing SystemVerilog routines
For Mastering Questa learning path, DPI chapter

The SV code in test.v calls imported function file_mem_size in c_tb.c
which reads a file and returns a value.

Compile & run on Linux with gui.csh and batch.csh

Compile & run on Windows with gui.cmd and batch.cmd

This works with Questa 10.7a on Linux and Windows, and probably eariler
versions too. However, C debug is only available on Linux.
